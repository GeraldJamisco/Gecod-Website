# GECOD Initiative Website

This repository contains the redevelopment of the official GECOD Initiative website.

## Overview

GECOD Initiative (Go Empower Communities for Development) is a community-based organization focused on youth empowerment, education, child protection, and sustainable community development.

This project transforms the existing PHP website into a secure, scalable Laravel-powered platform.

## Objectives

- Implement secure donation system (Stripe + Mobile Money)
- Build CMS for programs, blog, and gallery
- Enable volunteer management system
- Add multilingual support
- Improve SEO and global visibility
- Prepare platform for international donors

## Tech Stack

- Laravel (PHP Framework)
- MySQL
- Blade Templates
- Stripe API
- MTN Mobile Money (planned)
- Nginx / Apache
- Cloudflare (CDN)

## Core Features

- Admin Dashboard
- Programs Management
- Blog & News System
- Donation Processing
- Volunteer Application Form
- Secure Authentication
- SEO Optimization
- Multilingual Ready

## Deployment

Target hosting: International VPS  
SSL: Let's Encrypt  
CDN: Cloudflare
